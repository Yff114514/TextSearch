/**
    本项目实现的主要功能为加密文档的模糊搜索。
**/
    项目代码模块位于src文件夹。其中的若干文件夹功能如下：
    main：主程序，是运行项目的入口。
    ArticleDemo：存储文档集，预留存储100篇文档的空间。
    tools：存储一些重要功能模块。如：RSA加密体制、布隆过滤器、Bigram向量转换算法、局部敏感哈希算法LSH(结合Minhash）等,每一个模块都存储为一个java类，由主函数main调用。
    data：存储Minhash算法的随机参数。

    主函数KeywordSearch()的主要模块如下：
    KeyGen()       初始化密钥对，生成RSA公钥和私钥。
    getArticle()   从ArticleDemo文件夹导入文档集。
    indexBuild()   提取文档关键词，构建关键词-文档的倒排索引、布隆过滤器每个单元-关键词的索引。
    EncryptFile()  加密文档并存储。为简化流程仅存储在本地，比较好的方案是上传云端数据库。
    Search()       输入一个关键词并使用构造好的布隆过滤器进行匹配。理想的效果是：有一定的单词拼写容错率，输错2个字母也可以获得想要的文档。
    DecryptFile()  获得目标文档的编号，提取并解密，输出。
